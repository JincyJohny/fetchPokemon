# Fetch Pokemon

Fetch Pokemon is a Python program to fetch pokemon details based on rank.
## Installation

Use the package manager [pip](https://pip.pypa.io/en/stable/) to install requests.

```bash
pip install requests
```

## Usage

```python

# run the code
python3 ./mainFile.py   

# returns 25th pokemon's details                         
Enter the rank: 25

```
